# 待办事项列表 - Java版

一个简单的命令行待办事项管理程序。

## 功能特性
- ✅ 添加任务
- 📋 查看任务列表  
- ⚡ 标记任务完成状态
- 🗑️ 删除任务

## 项目结构
```
src/
├── Main.java      # 主程序
├── TodoItem.java  # 任务项类
└── TodoList.java  # 任务列表管理
```

## 使用方法
```bash
# 运行程序
java -jar TodoList.jar
# 或双击run_utf8.bat
```

## 运行示例
```
欢迎使用命令行待办事项列表！

请选择操作：
  add    - 添加任务
  list   - 列出所有任务  
  toggle - 标记/取消任务
  remove - 删除任务
  quit   - 退出程序
请输入命令:
```
